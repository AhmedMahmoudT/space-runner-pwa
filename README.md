# Space Runner

## A simple 3d space runner game built with Angular (PWA)

This project was made to show how Angular PWA helps us create websites that can be used offline and can act like a native app (desktop or mobile).  

## Technologies used

- [Angular](https://angular.io/) & [@angular/pwa](https://www.npmjs.com/package/@angular/pwa)
- [Three.js](https://threejs.org/)
- [Firebase Anonymous Auth & Realtime Database](https://firebase.google.com/docs)

### Live demo: [Space Runner](https://space-runner-pwa.vercel.app)
